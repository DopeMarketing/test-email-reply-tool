## [0.1.0] - 2026-03-26

### Added

#### Project Infrastructure
- Initial Next.js 15 project scaffold with App Router and TypeScript
- Supabase integration with authentication and database setup
- Tailwind CSS configuration with clean SaaS design system
- Vercel deployment configuration
- Environment variable structure for all integrations

#### Database Schema
- `auth_users` table for user authentication and roles
- `api_integrations` table for storing integration credentials and settings
- `email_filters` table for Gmail filtering rules by recipient
- `gmail_messages` table for incoming email data and metadata
- `conversation_threads` table for email thread tracking and context
- `hubspot_contacts` table for CRM contact synchronization
- `ai_response_drafts` table for generated email response storage
- `sms_approvals` table for tracking SMS approval requests
- `sms_responses` table for handling yes/no SMS replies
- `sent_emails` table for outbound email logging and tracking
- `automation_logs` table for system activity and error tracking
